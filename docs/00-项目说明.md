## 清理早期terraform 实验资源
```bash
cd terraform
# 列出terraform管理了哪些资源
terraform state list
# 预览即将删除的资源
terraform plan -destroy
# 销毁资源
terraform destroy
# 确认资源已销毁
terraform state list
```
## 重新初始化gcloud cli
```bash
rm -rf ~/.config/gcloud
# 登录gcloud cli
gcloud auth login --no-launch-browser
# 列出账号下的项目
gcloud projects list
# 设置默认项目
gcloud config set project gcp-training-20260303
# 设置默认区域 asia-east2 是香港
gcloud config set compute/region asia-east2
# 设置可用区
gcloud config set compute/zone asia-east2-a
# 查看default配置
gcloud config list
```
